Mini Desk Robot

A tiny desktop robot comes alive with an ESP32 
brain and a little OLED face. Built not for complex 
tasks but quiet companionship through shifting expressions. 
Its screen shows simple messages, reacting like it's thinking. 
Movement blends with visual feedback in subtle ways. 
Expression changes happen smoothly, frame by frame. 
Small details give it presence without noise. 
The build focuses on minimal parts doing more together. 
Emotion appears through timing, not realism. Each blink, each shift in pixels adds character slowly.

🔧 Features
Shows basic movements and facial looks
Uses oled display
Small size fits easily on desks
Built with ESP32

🛠️ Components Used
- ESP32
OLED I2C Display
- Jumper wires
- Breadboard

How It Works
A tiny screen lights up when the ESP32 sends signals, 
flashing faces or words. Depending on what gets added later - 
maybe a sensor or switch - the little machine might respond 
in new ways.
